# Sistema de Chamados TI
Projeto desenvolvido em C# no console com foco em prática de lógica de programação, CRUD, listas, métodos e Programação Orientada a Objetos.
## Funcionalidades
- Abrir chamados
- Listar chamados
- Alterar status
- Remover chamados
- Validação de IDs com TryParse
- Tratamento de erros básicos
## Tecnologias utilizadas
- C#
- .NET
- Console Application
## Conceitos praticados
- List<T>
- foreach
- if/else
- switch
- métodos
- classes e objetos
- validação de entrada
- return
- CRUD
## Objetivo do projeto
Praticar fundamentos de desenvolvimento backend utilizando C# e estruturação de sistemas simples no console.
## Como executar
1. Clone o repositório
2. Abra no Visual Studio Code ou Visual Studio
3. Execute o projeto
